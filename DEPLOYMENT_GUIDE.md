# CyberGenesis Land Mint DApp - Mainnet Deployment Guide

## Overview
This guide provides step-by-step instructions for deploying the CyberGenesis Land Mint DApp to the Internet Computer mainnet with all five canisters and 3D model assets.

## Prerequisites

- dfx CLI installed (version 0.15.0 or higher)
- Internet Computer wallet with sufficient ICP for cycles
- All 45+ 3D model assets (.glb files) ready in `assets/models/` directory
- Node.js and pnpm installed for frontend build
