# CyberGenesis - Руководство по развертыванию

## Обзор

CyberGenesis - это децентрализованное приложение на Internet Computer для управления виртуальными земельными участками с 3D-визуализацией.

## Требования

- Node.js 18+
- dfx (Internet Computer SDK)
- npm или yarn
